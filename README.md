# valtech-test

> A Valtech project test, using Vue.js, Saas and fractal.

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run all tests
npm test

# run fractal
cd src
fractal start --sync
```
